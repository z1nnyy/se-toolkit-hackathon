# Demo Script

## Goal

Show in under 2 minutes that the product is useful, working, and deployable.

## Suggested Voice-Over Flow

1. Introduce the problem:
   - "Students often need to walk to Cava just to check what is available. This product solves that with a Telegram bot and an admin panel."

2. Show the bot:
   - send `/start`
   - switch language
   - open one of the menu groups
   - open the sold-out view

3. Show the admin panel:
   - log in as admin
   - create or edit one dish
   - change availability or price

4. Return to the bot:
   - refresh the menu
   - show that the menu changed

5. Close with deployment:
   - mention that the whole project runs with `docker compose up --build`

## Recording Checklist

- speak clearly and keep the screen uncluttered;
- do not show secrets such as `BOT_TOKEN`;
- prepare the admin user in advance;
- prepare one dish to edit so the demo is quick;
- keep the browser and Telegram windows already open before recording.
